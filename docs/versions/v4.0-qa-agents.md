# Version 4.0 - QA 에이전트 시스템

**릴리스 날짜**: 2026-01

## 개요

게임 개발에 특화된 5종의 QA 에이전트를 도입한 업데이트입니다.
기술적 버그부터 밸런스, 보안까지 포괄적인 품질 검증이 가능해졌습니다.

---

## 주요 개선 사항

### 1. 5종 QA 에이전트

| 에이전트 | 역할 | 검증 영역 |
|---------|-----|----------|
| qa-tc | TC 작성 (GQA) | 테스트 케이스 설계 |
| qa-tech | 기술 버그 (TQA) | 코드 분석, 엣지케이스 |
| qa-balance | 밸런스 (FQA+DQA) | 경제, 수치, 재미 |
| qa-security | 보안 (CQA) | 어뷰징, 치팅 방지 |
| qa-release | 런칭 검수 (PQA+SQA) | 패치, 스토어 심사 |

---

### 2. QA 파이프라인

```
/eee_qa-full 실행 시:
1. qa-tech → 기술적 버그 분석
2. qa-security → 보안 취약점 분석
3. qa-balance → 밸런스 분석 (선택)
4. qa-tc → TC 문서 생성
5. 종합 리포트
```

---

### 3. 게임 특화 보안 검증

**검증 항목**:
- 돈복사 취약점
- 클라이언트 신뢰 문제
- 타임 조작
- 메모리 조작
- 경쟁 조건
